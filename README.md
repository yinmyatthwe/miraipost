


#  環境設定手順 (Environment Setup Guide)


まず、**GitHubのプロジェクトをクローン**してください。  

まだ **Angular** や **Node.js** はインストールしていないと思いますが、  
まずは自分のPCで **VS Code** を開き、クローンしたプロジェクトを開いてください。  
---


## 💡 バージョンの確認

VS Code のターミナルで、下のコマンドを1つずつ入力して  
**Node.js** と **npm** のバージョンを確認します。

```bash
node -v
npm -v
````

> 💡 もしインストールされていない場合、上のコマンドは動作しません。
> その場合は、まず **Node.js 20** をダウンロードしてインストールしてください。
> 👉 [https://nodejs.org/](https://nodejs.org/)

---

## ⚙️ パッケージのインストール

Node.js のインストール後、VS Code のターミナルで次のコマンドを順に実行します。

```bash
npm install -g @angular/cli
npm install
```

---

## 🚀 プロジェクトの起動

インストールが完了したら、次のコマンドでプロジェクトをビルド・起動します。

```bash
npm run build && npm start
```

または、Angular開発用サーバーで直接起動したい場合は：

```bash
ng serve --open
```

ブラウザが自動的に開き、「**Welcome**」ページ（初期のWebページ）が表示されればOKです。

---

## ⚠️ 注意

必ず **プロジェクトフォルダ内でターミナルを開いて** 作業してください。
